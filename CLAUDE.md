# Cyber Digest — project notes for Claude

## Deployment
- Live site: https://cyber-digest.pages.dev/
- Host: Cloudflare Pages
- Deployed from: `site/` subdirectory (NOT the repo root)

## Files that matter
| File | Purpose |
|------|---------|
| `site/index.html` | Homepage |
| `site/app.js` | All client-side behaviour (clock, KEV fetch, filters, ticker, search) |
| `site/styles.css` | All styles |
| `site/*.html` | Other pages (wire, cves, briefing, actor, about, etc.) |
| `digest.json` | Daily CVE/KEV/news data generated by GitHub Action |
| `digest.py` | Python script that generates digest.json |

## DO NOT edit
- `index.html` in the repo root — this is NOT served by Cloudflare
- Any file outside `site/` unless explicitly asked

## Architecture
- Static site, no build step, no bundler
- `app.js` fetches live data from CISA KEV and NVD APIs on page load
- `digest.json` is regenerated daily at 09:00 UTC via GitHub Actions
- No frameworks, no npm, plain HTML/CSS/JS

## Git workflow
- Branch: main
- Remote: https://github.com/CyberSecDigestDaily/cyber-digest.git
- Cloudflare auto-deploys on push to main
